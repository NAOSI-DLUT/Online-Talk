## PPT 渲染

本次的 slide 由 [sli.dev](sli.dev) 驱动支持，可以自行全局安装 slidev 后通过

```bash
slidev init.md
```

或本地安装后通过 `npm` 调用打开

启动后，访问 `http://localhost:3030` 进行查看
